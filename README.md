# TGUI
Standing for Terminal GUI

## ABOUT IT
Are you sick of having to handle your terminal all the time?  
Have you thought about using any sample media library but you are stuck on TTY's?  
Are you done handling yourself all the graphical user interfaces that has none to do with the agility of your terminal?  

Then **TGUI** is the best library available for you, let me explain:  

With a simple `tguiCreateWindow()` you can create a huge area of expandable graphical interface for your application!  
- "But I want my window to be transparent"  
No worries lil' dev, `tguiSetWinAttr(TGUI_ATTR_WIN_IS_OPAQUE, TGUI_FALSE)` sets your window as transparent one!  
- "Damn, that's really good ngl... but I want my window first to start blue then I'll change colors to red"  
No worries again! `tguiSetWinAttr(TGUI_ATTR_PXA_COLOR, TGUI_PIXEL_COLOR_BG_BLUE)` and later on `tguiUpdate(best_window_available, TGUI_ATTR_PXA_COLOR, TGUI_PIXEL_COLOR_BG_RED)` and voi'la! Changing colors!!  
- "But what if-"
No worries, brother or sister! We have everything you need for your terminal applications to **finally** look like it belongs to any graphical system operation! Even if you're stuck on old IBM 68000 pcs!
